About

This blog talks about computer technology. The old version can still be viewed at blog.nicolabs.net but will be deleted one day, since all new work is made here.

Making of

It is built with Jekyll on GitHub Pages. You can look at its source at github.com/nicolabs/nicolabs.net (but I bet you won’t find it interesting for now…).

Its theme is based on nandomoreirame’s “simplest” theme.

The code for estimating and displaying the time to read an article comes from atekihcan.github.io and github.com/renyuanz/leonids.

The main font ‘Open Sans’ is Apache-licensed.

The mastodon widget on the front page is from mastodon-timeline-widget.

The tags page has been possible thanks to this blog article.

Acknowledgment for third party content can be found here.
